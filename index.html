Transistions

  • Transition bedeutet Übergang
  • Bewegung von Interface-Elementen
  • Beispiel: im Smartphone zwischen Screens blättern

Transition im Code realisieren

  • Definiert mit zwei Zuständen, zwischen denen Animation abläuft
  • Start wird über Ereignis ausgelöst ("Event triggern")

Zustand A: Start

  • Transition-Eigenschaft mit mindestens zwei Werten für den Start definieren:
    • CSS-Eigenschaft
    • Zeit
.button {
background: white;
-web-transition: background 0.2s;
}

Zustand B: Ende

  • Die veränderten Werte für das Ende setzen
.button: hover {
background: red;
}

Beispiel: Transistion

Source-Code

.box { background: #2db34a; border-radius: 6px; cursor: pointer; height: 200px; line-height: 200px; text-align: center; transition-property: background; transition-duration: 1s; transition-timing-function: linear; width: 200px; } .box:hover { background: #ff7b29; }
